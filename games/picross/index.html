How to Play Picross Mosaic

Goal

Paint every cell the picture needs, and leave every other cell blank, using only the number clues along each row and column.

Clues

Each clue is a colored number giving the length of one unbroken run of that color in that row or column, in order. A row clued "3 red, 2 blue" has a block of 3 red cells, at least one gap, then a block of 2 blue cells, somewhere in that row.

Controls

Pick a color from the palette, then click (or tap) a cell to paint it - click a painted cell again to clear it. Toggle mark mode (the ✕ button, or right-click on desktop) to mark cells you're confident are blank, as a memory aid; marks don't count toward solving.

Winning

The puzzle solves itself automatically the moment the grid matches the picture. A row or column's clue dims once that line is fully satisfied.
