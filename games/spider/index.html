How to Play Spider

Goal

Clear all 104 cards by building eight complete King-to-Ace runs, each a single suit in unbroken descending order.

Setup

Two 52-card decks are combined and dealt into 10 tableau columns - the first 4 columns get 6 cards, the rest get 5. Only the top card of each column starts face-up. The remaining 50 cards sit in the stock. Before dealing, choose how many suits are in play: 1 suit is easiest (every card is a spade), 4 suits is the traditional, hardest variant.

Building

Any face-up card can be placed on a tableau card exactly one rank higher, regardless of suit. Click a card to select it (and any same-suit, sequentially-descending run beneath it on the same column), then click a destination column to move it there. Click the selected card again to cancel. A run can only be moved as a group if every card in it is the same suit and each is exactly one rank below the last - otherwise only the single top card of the column can move.

Empty Columns

Any card or valid group can be dropped onto an empty column - no King is required to start one.

Dealing More Cards

Click the stock pile to deal one card face-up onto every tableau column at once. This is blocked whenever any column is empty. The stock holds 5 deals total.

Completing a Sequence

Whenever the top 13 cards of a column form an unbroken King-to-Ace run of the same suit, they're automatically removed to the foundation.

Winning

Complete all 8 sequences to clear the table and win.
